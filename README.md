# Web-Project
Project Description:
Visual Pharma is a frontend project built using HTML, CSS, JavaScript, and Bootstrap to create a user-friendly online medicine browsing experience. This was my first college project, and it received great feedback for design and usability.

Key Features:

✅ Responsive Design: Works smoothly on mobile, tablet, and desktop.

✅ Interactive UI: Buttons, search, and cards made with JavaScript & Bootstrap.

✅ Clean & Simple Layout: Easy to navigate for all users.

✅ First Project Success: Well-received in college for creativity and design.

Technologies Used:
HTML | CSS | JavaScript | Bootstrap
